C#であることを活かすための機能は未実装ですが、このままでもLuaスクリプトの上位互換です
マシンクラフトVer.0.223以降はC#スクリプトを推奨し、Luaに関する保守を行いません

Lua用に用意した関数群がそのまま使えるはずなので、AutoPilotクラスに関しては
「Luaスクリプト解説.txt」を参照してください

多数のクラスを扱う場合は別ファイルに分けて別フォルダで管理したほうが良いでしょう
指定したフォルダ内の*.csファイルがコンパイル対象になります(サブフォルダを含む)
※名前が'_'で始まるファイルおよびフォルダは処理の対象外です

https://docs.unity3d.com/ja/current/Manual/ScriptingSection.html
を見ればスクリプトに関するほとんど全てがわかりますが、端から熟読するようなものではありません
「わからないことを検索すると公式リファレンスマニュアルにたどり着く」のが普通だと思います

サンプルには_TEST_Script(+Maneuver.cs)と_TEST_Library(+Graphics.cs)があり、
前者は単一クラスのシンプルな例で,後者はユーティリティクラスを含む例です
Luaスクリプトを書いたことがある人はold(lua)\trainer.txtとTrainer.csを見比べるのも良いでしょう
サポート関数やサンプルは随時追加予定です

----------------------------------------------------------------------------------------------------
使い方
----------------------------------------------------------------------------------------------------
1.BUILD-SystemMenu-ADVANCED-ScriptNameにスクリプト名を設定する(マシン名が"Tank_1"なら"Tank"がベスト)
※ライブラリフォルダがある場合は"Tank;Lib1;Lib2"のように';'で区切って付け足す
2.設定した名前のスクリプトファイル(*.cs)をUserData\_scripts\に作成する(Empty.csをコピーしてリネーム)
3.スクリプトファイルを編集する(保存すると自動でコンパイル開始)
4.コンパイルエラーがある場合はログに赤文字で表示される
5.コンパイルに成功すると緑文字でメッセージが表示され、スクリプトが実行される
6.実行時エラーが発生した場合はログに白文字で表示され、スクリプトが停止する
※F10でログウィンドウをON/OFFでき、チャットと同様にモディファイア+マウスホイールでスクロール
※F7キーでDEBUG実行/通常実行/停止が切り替わる(Luaと同じ)

----------------------------------------------------------------------------------------------------
注意点
----------------------------------------------------------------------------------------------------
・必ずGetUserName()がユーザー名を返すようにしなければばならない(危険なスクリプトの拡散を抑制)
・禁止ワードが含まれるスクリプトは使用できない(「禁止ワード一覧」参照)
・文字コードはUTF-8推奨(S-JISだと問題が起きるかも)
・ライブラリフォルダ内のスクリプトはコンパイラにロック(上書き禁止)される場合がある ※10秒程度で解除される

攻撃検出用マスク
----------------------------------------------------------------------------------------------------
ビットマスクで指定します
以下のように定義してSearchThreat(MASK_BULLET|MASK_MISSILE,500)のように書くと良いでしょう
const int MASK_BULLET = 1;  // 通常弾(Cannon)
const int MASK_SHELL = 2;   // 戦車砲弾(Cannon)
const int MASK_GRENADE = 4; // 榴弾(Cannon)
const int MASK_BLADE = 8;   // 刃(Sword)
const int MASK_PLASMA = 16; // プラズマ(Discharger)
const int MASK_LASER = 32;  // レーザー(Beamer)
const int MASK_MISSILE = 64;// ミサイル(Launcher)
const int MASK_ALL = 0xff;

----------------------------------------------------------------------------------------------------
AutoPilotクラスの公開関数一覧(C#用に追加したもののみ,未確認だがLuaでも使用可)
----------------------------------------------------------------------------------------------------
GameObject CreateSprite(string imageName)…スプライトを作成する(仮)
	imageName:UserData\_imagesフォルダにある画像ファイル(*.png,*.jpg)の名前(拡張子なし)
※サポート関数を用意していないため、操作するにはGetComponent関数を使用して必要なクラスを取得する必要がある

void LogC(string txt)…コンソール(左下の窓)に文字列を表示する
void LogC(float val)
void LogC(Vector3 vec)
	txt,val,vec:表示内容
※Practice(Test)専用なので、オンラインで使用してもキャンセルされる(無害)

bool IsDriving()…マシンを操作中か調べる(メニューorフリーカメラならfalse)

float GetFlightTime()…自機とステージの非接触期間(秒)を取得する

void FreeCursor(bool flag, bool stopDrive=true)…マウスカーソルのロック(非表示,中央に固定)を解除する
	flag:ON/OFF
	stopDrive:マシンの操縦を停止する(flag==trueの場合のみ有効)
bool IsCursorFree()…マウスカーソルのロック解除フラグを取得する
